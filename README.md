# Feedget

Aplicação implementada durante a 8º edição do NLW, a NLW Return. Dividido em 5 aulas, foi implementada uma aplicação de ponta a ponta usando React, TypeScript e outras ferramentas para auxiliar o desenvolvimento.

## Web
[Feedget - Web](https://nlw-return-impulse-d7iqqnysz-filipiandrader.vercel.app/)

## Backend
[Feedget - Backend](https://nlw-return-impulse-production-a3b8.up.railway.app/)

- Como usar o backend
```
@POST '/feedbacks'
Body: {
   "type":"type",
   "comment":"comment",
   "screenshot":"uri screenshot"
}
```
